Para fazer o build no IOS, executar os comandos
    -- para o Mac reconhecer o iphone, é necessário iniciar o Mac em modo seguro (ligue e imeditatamente pressione SHIFT, continue segurando até aparecer a tela de login)
    -- flutter build ios --release
    -- verifique os devices conectados: flutter devices
    -- flutter run --release -d "iPhone de Mariana"

Para buildar no android:
    -- para encontrar o nome/id do device: flutter devices
    -- depois para executar: flutter run -d 2107113SG
